## 符号说明

| 符号 | 含义 |
|------|------|
| $\mathbf{D}$ | 电位移矢量（$\mathbf{D} = \varepsilon \mathbf{E}$） |
| $\mathbf{E}$ | 电场强度 |
| $\mathbf{B}$ | 磁感应强度 |
| $\mathbf{H}$ | 磁场强度（$\mathbf{H} = \mathbf{B}/\mu$） |
| $\mathbf{J}$ | 电流密度 |
| $\rho$ | 自由电荷体密度 |
| $\varepsilon$ | 介电常数 |
| $\mu$ | 磁导率 |
