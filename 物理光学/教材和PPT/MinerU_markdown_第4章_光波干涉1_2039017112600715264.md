# 怎样利用普通光源获得相干光？

利用普通光源获得相干光的方法的基本原理是：把由光源上同一点同一次发的光设法分成两部分，然后再使这两部分叠加起来。由于这两部分光实际上都来自同一发光原子的同一次发光，它们满足相干条件而成为相干光。分开之后所走的路程不同，导致光波的相位不同，根据其差别分别产生干涉加强或者减弱。

# 怎样利用普通光源获得相干光？

利用普通光源获得相干光的方法的基本原理是：把由光源上同一点同一次发的光设法分成两部分，然后再使这两部分叠加起来。由于这两部分光实际上都来自同一发光原子的同一次发光，它们满足相干条件而成为相干光。分开之后所走的路程不同，导致光波的相位不同，根据其差别分别产生干涉加强或者减弱。
