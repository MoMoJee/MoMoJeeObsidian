# 复折射率：

考虑到介质对光的吸收，介质的光学性能需要由两个参数来描述，即折射率n和吸收系数。

![image](images/8c5546de6090e6b8ce425553eee5502dcf03ddf40f989fd9ae39fab500bf1d1c.jpg)


无吸收透明介质，沿x方向传播的平面波函数：

$$
\widetilde {E} (x, t) = A _ {0} e ^ {- i (\omega t - k x)} = A _ {0} e ^ {i k x} e ^ {- i \omega t}
$$

吸收介质中，光强随传播距离x衰减：

$$
I (x) = I _ {0} e ^ {- \alpha \cdot x}
$$

光强和振幅的关系：

$$
I (x) = A ^ {2} (x) \Rightarrow A (x) = A _ {0} e ^ {- \frac {\alpha}{2} x}
$$

所以在吸收介质中的波函数可以写成：

![image](images/34700de27ba9372711f4e76f51b894c0b48f2fae01e96e0614ad5a00ed0d7d08.jpg)


k为复波矢： ${ \tilde { k } } = k + i { \frac { \alpha } { 2 } }$ 

实波矢和折射率的关系：

$$
k = \frac {2 \pi}{\lambda} = \frac {\omega}{\nu} = \frac {\omega}{c / n} = \frac {\omega}{c} n
$$

所以：

$$
\widetilde {k} = \frac {\omega}{c} \left(n + i \frac {c \alpha}{2 \omega}\right) = \frac {\omega}{c} \widetilde {n}
$$

其中 为复折射率，n n~ ~  $n + i \frac { c \alpha } { 2 \omega } = n \big ( 1 + i \kappa \big ) , \kappa = \frac { c \alpha } { 2 n \omega } =$ 0n4它的虚部反映了介质的 吸收， 称为衰减系数。

# 正常色散：

对光波透明的介质，其折射率n随波长λ的增加而减小，色散曲线(n-λ关系曲线)如下图所示，称为正常色散。

![image](images/d84f4ef06e5ebc25850c7bfaa83da8b579b6e78d3faf8b5454c2201a8347a9be.jpg)


法国数学家A.L.柯西于首先给出了正常色散的经验公式，称柯西公式：

$$
n = A + \frac {B}{\lambda^ {2}} + \frac {C}{\lambda^ {4}}
$$

在可见光波段，在很多材料中，柯西公式相当准确地反映了实际色散曲线。当考察的波长范围不太大时，只要取柯西公式的前两项就够了：

$$
n = A + \frac {B}{\lambda^ {2}}
$$

$$
\frac {d n}{d \lambda} = - 2 \frac {B}{\lambda^ {3}}
$$

表明紫段的色散效应比红段的要大。

# 反常色散：

在介质对光有强烈吸收的波段内(吸收带)，折射率随波长的增加而减小，色散率dn／dλ＞0，这与正常色散相反，故称反常色散。

![image](images/caf6c83e181517c30d337d883e36486ea9f0cb5f5d42a432f03255956119064f.jpg)



图9.4吸收带附近的反常色散


在可见光区域内色散是正常色散曲线（PQ段）满足柯西公式。若向红外区域延伸，并接近吸收带时，色散曲线开始与柯西公式偏离见。在吸收带内因光极弱很难测到折射率的数据，过了吸收带，色散曲线又恢复正常的形式，并满足柯西公式，不过式中的常数A，B，C等换为新的数值了。F.-P.勒鲁于1860年首先在碘蒸气棱镜内观察到反常色散现象 。

R.W.伍德于1904年利用交叉棱镜法成功地显示出钠蒸气在可见光波段内的反常色散。

![image](images/9aee01fce2caa44d0803d98ffc4bb80d12772f1107c0a195dd74c601c504a620.jpg)



图9.5观察钠蒸气反常色散的实验装置—交叉棱镜法


令一束光从水平狭缝S1穿出，经透镜L1变为平行光，再经过透镜L2聚焦在分光仪的竖直狭缝S2上，当钢管V未加热时，其内只有均匀气体，光线经过它时不发生偏折，在分光仪上形成一水平光谱带，当钠被蒸发时，在管内形成下部密度大上部密度小的水平钠蒸汽柱，它和一个棱边在上（与管轴垂直）底部在下的“棱镜”等效。由于管V内蒸汽的色散作用，不同波长的光不同程度的向下偏折，在钠的吸收线附近，分光仪焦面上的水平光谱带被严重扭曲和割断，变成图所示的样子，这种现象叫做反常色散。

# 经典色散理论

概述：光与物质的相互作用导致了色散效应。

经典电磁场理论表明，介质中单色电磁波的传播速度：

$$
\nu = \frac {1}{\sqrt {\varepsilon \varepsilon_ {0} \mu \mu_ {0}}} = \frac {c}{\sqrt {\varepsilon \mu}} \approx \frac {c}{\sqrt {\varepsilon}} \qquad (\mu \approx 1)
$$

介质的光学折射率：

$$
n = \sqrt {\varepsilon}
$$

介质的相对介电常数，联系着宏观上的介质极化强度P与电场强度矢量E：

$$
P (t) = \varepsilon_ {0} \chi E (t), \varepsilon = 1 + \chi
$$

其中称为介质的电极化率。

# ●波拍的群速度

平面单色光波； $\widetilde { U } ( x , t ) = A e ^ { i ( k x - \omega t ) }$ 对于空间某一点 $( x _ { 0 } , t _ { 0 } ) \vert ^ { t }$ 的运动状态由它的相位 $( k x _ { 0 } - \omega t _ { 0 } )$ 来刻画，经历dt时间，该运动状态传播到 $( x _ { 0 } + d x )$ ；显然，两个相位值相等。

![image](images/b8110938034c4daa58b533ca66f6dc7ba750790505ff4038f96cf13fea34c359.jpg)


$$
k (x _ {0} + d x) - \omega (t _ {0} + d t) = k x _ {0} - \omega t _ {0}
$$

波的速度为单位时间里运动状态的传播距离：

$$
\nu_ {p} = \frac {d x}{d t} = \frac {\omega}{k}
$$

这个速度的定义是运动状态的传播，或者说是相位的转播速度，故称为相速度。

对于理想的单色光，相位的传播速度和能量的传播速度完全一致，故没有必要特意指出其为相速度。但是在非单色光的情况下会出现一个速度概念—群速度，此时才有必要将单色光波的速度称为相速度，以示区别。
