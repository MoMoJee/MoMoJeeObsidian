# *电磁场边值关系

电磁场边值关系由麦克斯韦积分方程给出，其反映了电磁场在两种介质分界面处的突变的规律。

$$
\left\{ \begin{array}{l} \oint_ {L} \vec {E} \cdot d \vec {l} = - \frac {d}{d t} \iint_ {S} \vec {B} \cdot d \vec {S} \\ \oint_ {L} \vec {H} \cdot d \vec {l} = I _ {f} + \frac {d}{d t} \iint_ {S} \vec {D} \cdot d \vec {S} \\ \iint_ {S} \vec {D} \cdot d \vec {S} = Q _ {f} \\ \iint_ {S} \vec {B} \cdot d \vec {S} = 0 \end{array} \right.
$$

$$
\left\{ \begin{array}{l l} \overrightarrow {n} \times (\overrightarrow {E} _ {2} - \overrightarrow {E} _ {1}) = 0 \\ \overrightarrow {n} \times (\overrightarrow {H} _ {2} - \overrightarrow {H} _ {1}) = \alpha \\ \overrightarrow {n} \cdot (\overrightarrow {D} _ {2} - \overrightarrow {D} _ {1}) = \sigma \\ \overrightarrow {n} \cdot (\overrightarrow {B} _ {2} - \overrightarrow {B} _ {1}) = 0 \end{array} \right.
$$

$$
\left\{ \begin{array}{l} \oint_ {L} \vec {E} \cdot d \vec {l} = - \frac {d}{d t} \iint_ {S} \vec {B} \cdot d \vec {S} \\ \oint_ {L} \vec {H} \cdot d \vec {l} = I _ {f} + \frac {d}{d t} \iint_ {S} \vec {D} \cdot d \vec {S} \\ \iint_ {S} \vec {D} \cdot d \vec {S} = Q _ {f} \\ \iint_ {S} \vec {B} \cdot d \vec {S} = 0 \end{array} \right.

1. 正入射时。 $\mathbf { n _ { 1 } } { < } \mathbf { n _ { 2 } } ,$ 界面反射有相位突变，有半波损失；$\mathbf { n _ { 1 } } \mathrm { > n } _ { 2 } .$ ,界面反射没有相位突变，即没有半波损失。

2. 掠射时。无论 $\mathbf { n _ { 1 } } \mathrm { > n } _ { 2 }$ 还是 $\mathbf { n _ { 1 } } \mathrm { < n } _ { 2 } ,$ ，界面反射均有半波损失。
