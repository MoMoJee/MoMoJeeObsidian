# 论文大纲：系统数据处理与识别性能评估

## 2.4 系统数据处理与输出层架构
本节主要阐述光强度信号从原始采集到最终语义输出的完整数据流向，重点在于提取真实按压动作、运用多模型进行特征识别，以及经由大语言模型（LLM）纠错输出的设计流。

### 2.4.1 时序信号的波动检测与预处理
*   **按压事件定位**：描述系统如何通过监测物理信号的方差与阈值，识别光强“平静-下降-上升-平静”的完整衰减与恢复周期，从而精准截取有效数据段。
*   **数据清洗与分段**：陈述通过基线校正与标准化消除环境静态偏移的方法。介绍如何利用重叠滑动窗口（Sliding Window）机制将长时序数据按固定步长切割，为网络输入做准备。

### 2.4.2 基于多模型特征提取的协同识别
*   **张量重组与维度适配**：解释为了适配一维卷积标准，将二维时序数据通过维度置换转化为三维张量的过程。
*   **异构神经网络映射**：
    *   **多尺度特征提取CNN**：说明利用大、中、小三个尺寸的感受野捕捉按压突变与整体趋势。
    *   **曲线凹陷专用CNN**：说明系统如何针对光纤弯曲独有的光强“凹陷”指纹进行深层特征强化。
    *   **ResNet残差网络**：利用跳跃连接结构提取高阶语义特征，防止梯度退化。

### 2.4.3 置信度决策与智能语义输出
*   **连续性置信度累积**：阐述单次独立识别通过累计多个采样窗口的置信度得分，结合连续预测的最长次数，选定最终字符的防抖动机制。
*   **基于大模型的语义纠错翻译**：介绍硬件识别缓冲后的高置信字符序列如何通过 Tool Calling 机制送入LLM，将纯大写英文字母还原并进行中英文（直译/意译）转化，最后触发语音模块播报的过程。

## 4.4 识别算法性能评估
本节旨在通过标准化数据集测试系统的鲁棒性，并通过科学地分析错分字符类别，论述复合深度学习模型在光纤传感盲文识别上的应用价值。

### 4.4.1 数据集构建与自适应增广
*   **实验样本采集**：明确实验数据的来源（3名志愿者，26个字母，每类重复采集，共计3900个标准数据样本）。
*   **滑窗增广机制**：阐述基于滑动窗口切分策略如何有效地扩充出供模型训练与识别的庞大数据量。

### 4.4.2 系统识别精度与混淆误判分析
*   **整体准确率概况**：展示系统98.31%的平均准确率，并标出最高（N）与最低（S）的极端值。
*   **空间结构导致的模式相似性探讨**：详细分析 D 与 E、H 与 J 等结构相似或高度对称盲符，其引发的光强时域波形高度同质化问题，解释其错误率稍高的物理内因。

### 4.4.3 多模型融合机制的性能增益
*   **协同判断的抗扰动效果**：讨论前述三大异构网络在特征提取上的互补作用，以及置信度融合机制如何有效地抵消了由于志愿者手动按压力度不均、时间长短不一引入的实验误差。
